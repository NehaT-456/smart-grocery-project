# Smart Grocery Project

A Flask and MySQL based e-commerce web application for online grocery shopping.

## Features

### User
- User Registration & Login
- Browse Products
- Search Products
- Add Products to Cart
- Checkout System
- Order History

### Admin
- Admin Login
- Product Management
- Add/Edit/Delete Products
- View Customer Orders
- Update Order Status
- Dashboard Statistics

## Technologies Used
- Python
- Flask
- MySQL
- HTML
- CSS
- Bootstrap
- JavaScript

## Installation

```bash
pip install -r requirements.txt
python app.py
